Update rules script

This script is a lightweight example of how to update the same firewall rule on all your domains.
 1# If you want to run this script, move it one folder above (next to cf.py)
 2
 3import os, dotenv
 4
 5from cf_rules import Cloudflare
 6
 7dotenv.load_dotenv(".env")
 8
 9rule_to_update = "Handle Bad Bots"
10
11cf = Cloudflare("expressions_main")
12cf.auth(os.environ.get("EMAIL"), os.environ.get("KEY"))
13
14# Export all rules from the main domain
15s = cf.export_rules("main-domain.com")
16
17# TODO Edit your rules before updating them back to Cloudflare
18
19# Update your rule for all domains
20for domain in cf.domains:
21    s = cf.update_rule(domain.name, rule_to_update, "Bad Bots")
22    print(s)

Note

If you export the rules from a domain, you will not be able to update them again if the text file hasn’t changed. This is a Cloudflare limitation.
